#### 如何运行 - How to run
```
# 克隆代码 - clone code
$ git clone https://github.com/jgsrty/jgsrty.github.docs.git
$ cd jgsrty.github.docs
```
#### 安装依赖 - Install dependencies
```
# 安装 - install
$ npm install 

# 本地运行 - dev
$ npm run dev

# 发布线上 - prod
$ npm run sh
```
